Mocha Java

Tak sepenuhnya kopi ini berasal dari Jawa. Pasalnya, mocha Java merupakan campuran bijih kopi dari Yaman dan Jawa. Sebuah perpaduan yang mempunyai karakteristik komplementer. Di Indonesia, kopi ini biasa tumbuh di daerah perbukitan lereng Menoreh, Kulonprogo, Yogyakarta. Terdapat dua jenis, yakni Mocha Java single dan double yang diproses dengan cara berbeda.
